Nurse SRNH CMS - Installation Guide
===================================

1) Start web server (Apache/XAMPP or PHP built-in)
   - Option A: Open installer\run_installer.bat
   - Option B: run `php -S 127.0.0.1:8080 -t c:\nurse_srnh`

2) Open installer page
   - http://127.0.0.1:8080/install.php

3) Fill DB connection and admin credentials
   - Installer will create:
     - Database
     - Tables (admins, settings, content_items, personnel, personnel_connections)
     - Default settings
     - Sample showcase content (achievements, directory, links, activity)
     - Upload directories (uploads/content, uploads/personnel)

4) After successful install
   - View website: /nurse_srnh/index.php
   - View achievements quickly: /nurse_srnh/managed-content.php?section=achievements
   - Admin login: /nurse_srnh/admin/login.php

5) Security
   - Remove or block access to install.php after successful installation.
   - Ensure `uploads/` is writable by web server user.

Generated installer files:
- install.php (web installer)
- installer/schema.sql (DB schema)
- installer/run_installer.bat (quick start)
